# MVP-083: Style the responsive frontend

## Description
Add local CSS for a clear usable form and result at narrow mobile and common desktop widths.

## Acceptance Criteria
- No horizontal scrolling occurs at 320 CSS pixels.
- Controls have visible focus states and sufficient target size.
- Success, error, disabled, and loading states are visually distinct without color alone.
- Testing: automated style assertions plus screenshots at 320px and 1280px show no clipping or overlap.
