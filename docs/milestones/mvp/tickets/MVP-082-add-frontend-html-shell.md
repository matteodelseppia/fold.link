# MVP-082: Add the frontend HTML shell

## Description
Create the static single-page HTML structure with product heading, URL form, submission control, result region, and error region.

## Acceptance Criteria
- Every input has a programmatic label and help text.
- Result and error regions support accessible announcements.
- No external runtime asset or analytics dependency is required.
- Testing: parse the served HTML and assert the required landmarks, labels, IDs, and asset references exist.
